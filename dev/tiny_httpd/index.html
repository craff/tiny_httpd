Library tiny_httpd

This library exposes the following toplevel modules:

  • Tiny_httpd
  • Tiny_httpd_buf Simple buffer.
  • Tiny_httpd_dir Serving static content from directories
  • Tiny_httpd_html HTML combinators.
  • Tiny_httpd_html_
  • Tiny_httpd_server HTTP server.
  • Tiny_httpd_stream Byte streams.
  • Tiny_httpd_util
